# 神雕农机完善方案 — 对标 RB Global（网站）+ tractorpool（小程序）

> 调研日期：2026-07-04
> 对标网站：https://rbglobal.com/（全球最大二手设备交易集团）
> 对标小程序：https://www.tractorpool.com/（欧洲最大二手农机平台，月访问400万+）

---

## 一、网站完善方案（对标 RB Global）

### 1. 差异分析

| 维度 | RB Global | 神雕农机现状 | 差距 |
|------|-----------|-------------|------|
| **交易模式** | 多渠道：拍卖+商城+政府 surplus+ salvage | 单一商城模式 | ❌ 缺少拍卖、竞价等交易模式 |
| **设备检验** | IronClad Assurance 检验报告 | 无 | ❌ 缺少设备检测报告体系 |
| **数据洞察** | Rouse 设备估值+市场基准+租金洞察 | 有AI估值+日报套利分析 | ⚠️ 有基础但缺深度行业报告 |
| **售后服务** | SmartEquip 零配件采购+目录+电商 | 无 | ❌ 缺少零配件模块 |
| **物流运输** | Veritread 专业运输 | 有物流信息页但无预订 | ⚠️ 有入口无服务闭环 |
| **行业方案** | 按行业（建筑/农业/运输/政府）定制 | 无行业区分 | ❌ 缺少行业解决方案页 |
| **信任体系** | 品牌背书+检验报告+交易保障 | 有卖家认证但缺交易保障 | ⚠️ 基础有，深度不够 |
| **导航结构** | 清晰的解决方案/服务/洞察/品牌分类 | 8个平铺导航项 | ⚠️ 需优化信息架构 |

### 2. 具体改进项

#### P0 — 必须做（1-2周）

**① 导航重构 + 文案修改**

```
当前导航：首页 | 设备市场 | 品牌▼ | 品类▼ | 博客 | 情报 | 物流 | 关于

改为：
首页 | 买农机▼ | 卖农机▼ | 市场洞察▼ | 服务支持▼ | 关于我们
                │           │              │              │
                ├─浏览设备   ├─发布产品     ├─每日日报     ├─物流方案
                ├─品牌馆     ├─发布指引     ├─套利分析     ├─检验服务
                ├─品类专区   ├─卖家中心     ├─行业报告     ├─零配件
                └─买家需求   └─积分管理     └─市场情报     └─联系我们
```

**② 新增"卖农机"入口到导航栏**
- 在导航配置 `navigation.ts` 中新增卖农机下拉菜单
- 直接指向 `/seller/products/new`
- 未登录用户点击跳转登录页

**③ 首页 Hero 区改造**
- 主标题改为：`神雕农机全球交易`
- 副标题改为：`智能估价 · 跨境物流 · 安全交易`
- CTA 按钮改为：`立即选购` + `发布卖机`

**④ 产品卡片增强**
- 增加卖家信息展示（公司名+国家）
- 增加设备状态标签（已检验/有视频/可出口）
- 无图产品显示品牌Logo占位图

#### P1 — 应该做（2-4周）

**⑤ 设备检验报告模块**
- 新建 `/api/inspection/[productId]` 接口
- 产品详情页增加"检验报告"标签页
- 检验项：发动机/液压/传动/外观/轮胎（5大类20项）
- 评分制：A/B/C/D 四级
- 有检验报告的产品在卡片上加绿色"已检验"标签

**⑥ 零配件专区**
- 新建 `/parts` 页面
- 复用产品列表组件，category 筛选为"配件"
- 支持按设备型号反查兼容配件

**⑦ 行业解决方案页**
- 新建 `/solutions/[industry]` 动态路由
- 覆盖：牧草/粮食/经济作物/畜牧 4大场景
- 每个方案页：推荐设备+典型配置+价格区间+案例

**⑧ 物流服务升级**
- 物流页增加"在线询价"表单
- 输入：设备尺寸+起运港+目的港+运输方式
- 返回：预估运费+运输周期+所需文件
- 对接已有的港口匹配模块

#### P2 — 可以做（1-2月）

**⑨ 数据洞察中心**
- 新建 `/insights` 页面
- 整合日报数据为可视化图表（价格趋势/品牌热度/品类分布）
- 按月发布"二手农机市场报告"（自动生成PDF）
- 对标 RB Global 的 Rouse Equipment Insights

**⑩ 在线拍卖功能**
- 新建拍卖模块 `/auction`
- 定期拍卖会（月度/季度）
- 保证金制+实时竞价
- 对接支付系统

**⑪ 运输追踪系统**
- 对接货代API
- 卖家/买家可在网站查看运输状态
- 类似快递追踪体验

## 二、小程序完善方案（对标 tractorpool）

### 1. 差异分析

| 维度 | tractorpool | 神雕农机小程序现状 | 差距 |
|------|-------------|-------------------|------|
| **搜索筛选** | 制造商/品类/价格/年份/国家/邮编半径/仅带图/仅带视频 | 关键词+4个下拉筛选 | ❌ 筛选维度严重不足 |
| **品类体系** | 28主分类+多级子分类，14万+商品 | 9个品类，80+商品 | ⚠️ 品类少但需完善子分类 |
| **收藏功能** | 收藏列表+保存搜索+关注卖家 | 有收藏页面但功能弱 | ⚠️ 需增强 |
| **信任体系** | Trusted Dealer认证+数字合同+支付保障 | 无 | ❌ 缺少信任机制 |
| **价格展示** | 净价/含税价切换+询价制+40+货币 | 仅人民币价格 | ⚠️ 需增加货币切换 |
| **买家功能** | 发布求购需求+管理询价 | 无 | ❌ 缺少求购/询价功能 |
| **零配件** | 独立配件模块 | 无 | ❌ 缺少配件入口 |
| **租赁功能** | 独立租赁模块 | 无 | ❌ 可选增加 |
| **多语言** | 24种语言自动翻译 | 仅中文 | ⚠️ 小程序可不做多语言 |
| **通知推送** | 新品通知+保存搜索推送 | 无 | ❌ 缺少推送体系 |

### 2. 具体改进项

#### P0 — 必须做（1-2周）

**① 文案修改（app.json）**
```json
tabBar 修改：
  "🔍 找农机" → "🔍 买农机"
  "✏️ 发布"   → "✏️ 卖农机"
  导航标题 "神雕农机" → "神雕农机全球交易"
```

**② 搜索筛选增强（list页）**

当前只有4个筛选tab，改为 tractorpool 式的全面筛选：

```
新增筛选项：
├─ 品类（已有，增加子分类）
├─ 品牌（已有）
├─ 价格区间（已有，增加滑块选择器）
├─ 年份范围（新增：2000-2026滑块）
├─ 产地/位置（新增：省份选择）
├─ 仅看有图（新增：开关）
├─ 仅看有视频（新增：开关）
├─ 仅看已检验（新增：开关，对接检验报告）
├─ 排序方式（已有，增加：价格↑/↓/最新/套利空间↓）
└─ 买家需求匹配（新增入口）
```

**③ 产品卡片增强**

当前卡片只显示：图片+品牌型号+价格

增加展示：
```
┌─────────────────────────────┐
│  [产品图片]    🔥套利34%     │
│                            │
│  John Deere 5045E          │
│  2018年 | 55HP | 河北       │
│  ¥15.0万  ≈$2,080          │
│  ✅已检验  🎬有视频  🚢可出口 │
│  ─────────────────────────  │
│  🏪 神雕农机认证卖家         │
└─────────────────────────────┘
```

**④ 买家需求匹配功能（新增页面）**

在"买农机"tab顶部增加需求匹配入口：

```
┌──────────────────────────────┐
│  🎯 智能匹配 — 告诉我你要什么  │
│                              │
│  我的农场：[___]亩            │
│  种植作物：[小麦▼]           │
│  需要机型：[拖拉机▼]          │
│  预算范围：[5]万-[15]万       │
│                              │
│       [ 获取推荐 ]            │
└──────────────────────────────┘
```

提交后调用 `/api/buyer-match/suggest`：
- 根据作物→推断品类
- 根据预算→筛选价格区间
- 结合日报情报→市场行情建议
- 返回3-5条精准推荐+采购建议

**⑤ 收藏/关注功能增强**
- 产品详情页增加"收藏"按钮（已有页面，需接线）
- 卖家详情页增加"关注卖家"按钮
- 新增"保存搜索条件"功能（下次打开自动恢复筛选）

#### P1 — 应该做（2-4周）

**⑥ 卖家信任体系**
- 卖家详情页增加认证标识：
  - 🏪 实名认证
  - 📋 营业执照认证
  - 🚚 可提供物流
  - 🔧 可提供售后
- 产品详情页显示卖家评分（如有交易记录）

**⑦ 询价/联系卖家流程**
- 产品详情页增加"询价"按钮
- 弹出询价表单：姓名+电话+留言
- 询价记录保存到"我的询价"
- 卖家在"卖家中心"看到询价列表

**⑧ 零配件入口**
- 首页增加"配件商城"入口
- 列表页支持按"整机/配件"切换
- 配件支持按兼容机型筛选

**⑨ 推送通知体系**
- 关注卖家后，该卖家新上架产品推送
- 保存搜索条件后，有新匹配产品推送
- 订阅消息模板（微信小程序订阅消息）

#### P2 — 可以做（1-2月）

**⑩ 租赁功能**
- 新增"租农机"入口
- 卖家发布产品时可选择"出售/出租/都可"
- 租赁价格：日租/月租
- 租赁流程：预约→支付押金→发货→归还→退押金

**⑪ 数字合同**
- 交易达成后生成电子合同
- 双方签字确认
- 合同存档可随时查看

**⑫ 多货币价格展示**
- 小程序设置页增加货币选择
- 产品价格同时显示人民币+所选货币
- 汇率每日更新（已有汇率数据源）

## 四、技术依赖关系

```
买家需求匹配 API ──→ 网站需求表单 + 小程序需求表单
     │
     └──→ 日报数据解析器（已有）
     └──→ 产品数据库查询（已有）

设备检验报告 ──→ 产品详情页标签页（网站+小程序）
     │
     └──→ 新建 Inspection 模型（Prisma migration）
     └──→ 新建 /api/inspection/[productId] 路由

卖家信任体系 ──→ 卖家详情页改造
     │
     └──→ Seller 模型增加认证字段（已有部分）
     └──→ 新建 /api/seller/[id]/profile 路由

推送通知 ──→ 微信订阅消息
     │
     └──→ 新建 /api/notifications/subscribe 路由
     └──→ 定时任务检查匹配产品
```
