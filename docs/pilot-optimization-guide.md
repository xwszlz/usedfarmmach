# 产品封面图试点优化指南

## 🎯 试点目标

通过优化3个重点产品的封面图，建立标准化的选图流程，提升用户第一印象和点击率。

| 序号 | 产品 | ID | 年份 | 状态 |
|------|------|----|------|------|
| 1 | 东洋 Beet Harvester | `cmpdknl8s00e511kwiy4tzjax` | 2018 | 良好 |
| 2 | 克拉斯 860 | `cmpdknii2000111kwcau06hey` | 2024 | 优良 |
| 3 | 约翰迪尔 7250 | `cmpdknj9v004b11kwqvki68wr` | 2013 | 一般 |

## 🛠️ 工具清单

1. **评审工具**：`public/pilot-cover-review.html` - 专门为3个试点产品设计的图片评审页面
2. **更新脚本**：`scripts/update-cover-images.ts` - 数据库更新脚本
3. **选择文件**：`cover-selections.json` - 记录您的选择
4. **原始分析**：`product_image_analysis.json` - 所有产品的图片数据

## 📋 选图标准（请严格遵循）

### ✅ 优先选择
- **整机展示** - 完整展示农机全貌
- **品牌标识可见** - 清晰显示品牌Logo
- **光线充足** - 图片明亮，细节清晰
- **角度适宜** - 正面或斜45度角
- **背景整洁** - 无杂乱干扰

### ❌ 避免选择
- **局部细节图** - 引擎、轮胎等局部
- **模糊暗光** - 光线暗，模糊不清
- **背景杂乱** - 分散注意力
- **无品牌标识** - 找不到品牌信息
- **角度怪异** - 难以理解结构

## 🚀 操作步骤

### 第1步：评审图片

1. 在浏览器中打开 `public/pilot-cover-review.html`
2. 查看3个产品的所有图片（每个产品10张）
3. 点击小图可以放大查看原图
4. 按照选图标准评估每张图片

### 第2步：选择最佳封面图

1. 在每个产品的图片网格中，点击最佳图片（会显示绿色边框）
2. 系统会自动记录您的选择
3. 填写选择理由（在页面底部的表格中）

### 第3步：记录选择

1. 打开 `cover-selections.json` 文件
2. 将您的选择填入文件：
   ```json
   [
     {
       "productId": "cmpdknl8s00e511kwiy4tzjax",
       "bestImageNumber": 5,
       "reason": "整机展示清晰，品牌标识可见，光线充足"
     }
   ]
   ```

### 第4步：更新数据库

1. 在项目根目录运行更新脚本：
   ```bash
   npx tsx scripts/update-cover-images.ts
   ```
2. 脚本会读取 `cover-selections.json` 并更新数据库
3. 查看运行结果，确认更新成功

### 第5步：验证效果

1. 重新启动网站或等待缓存更新
2. 访问网站查看3个试点产品的封面图
3. 确认封面图已按您的选择更新

## 🔧 技术说明

### 数据库更新逻辑
- 最佳图片：设置 `sortOrder = 0` 和 `isPrimary = true`
- 其他图片：设置 `sortOrder = 1-9` 和 `isPrimary = false`
- 前端显示：优先显示 `sortOrder=0` 的图片

### 图片存储结构
```
阿里云OSS:
  uploads/products/{productId}/1.jpg
  uploads/products/{productId}/2.jpg
  ...
  uploads/products/{productId}/10.jpg
```

### 缓存注意事项
- 浏览器可能会缓存图片，更新后可能需要清除缓存
- Vercel CDN 可能有缓存，更新后可能需要等待几分钟
