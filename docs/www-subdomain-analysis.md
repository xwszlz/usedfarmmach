## ⚙️ WWW 子域名配置建议

### **选项 A：添加 CNAME 记录（推荐）**

为 `usedfarmmach.cn` 域名添加 `www` 子域名：

1. **阿里云 DNS 控制台** → **云解析 DNS**
2. 找到 `usedfarmmach.cn` 域名
3. **添加记录**：
   - **记录类型**：`CNAME`
   - **主机记录**：`www`
   - **记录值**：`usedfarmmach.cn`（注意：不带 `http://` 或末尾斜杠）
   - **TTL**：保持默认（600秒）

**优点**：
- CNAME 记录会自动跟随根域名的 IP 变化
- 如果 Vercel IP 变更，无需手动更新
- 符合最佳实践

### **选项 B：添加 A 记录（备选）**

作为备选方案，也可以为 `www` 子域名单独添加 A 记录：

1. **阿里云 DNS 控制台** → **云解析 DNS**
2. 找到 `usedfarmmach.cn` 域名
3. **添加记录**：
   - **记录类型**：`A`
   - **主机记录**：`www`
   - **记录值**：`76.76.21.21`
   - **TTL**：保持默认
